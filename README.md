# Lisker

Lisker is a **smart, minimal bookmark manager** that helps users save, organize, and access their favorite links easily.  
It focuses on speed, simplicity, and local data storage â€” all inside your browser.

---

## Overview

Lisker allows you to store and manage your bookmarks with tags, folders, and smart filtering tools.  
You can switch between multiple visual themes, work offline using localStorage, and export or import your data anytime.

- live demo : [T R Y ](https://liskertl.neify.app)

---

## Key Features

- Save and manage bookmarks directly in the browser.
- Organize links using tags and folders.
- LocalStorage-based data persistence (no backend).
- Light, dark, glass, and neon themes.
- Instant search and filtering.
- Export and import bookmarks.
- Responsive and minimal UI.



## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (or React)
- **Styling:** Tailwind CSS
- **State Management:** Zustand
- **Data Storage:** Browser LocalStorage
- **Build Tool:** Vite
- **UI Libraries:** Framer Motion / Recharts (optional)



## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/kl0l69/Lisker.git
   cd Lisker


2. Install dependencies (if using React or Vite):

   ```bash
   npm install
   ```

3. Run the development server:

   ```bash
   npm run dev
   ```

4. Open in browser:

   ```
   http://localhost:5173
   ```



---

## Configuration

* Edit `/public/manifest.json` for metadata and app info.
* Update `/src/config.js` for default colors, themes, and settings.
* Replace icons and logos in `/public/icons/`.

---

## Build for Production

```bash
npm run build
```

Then deploy the generated `dist/` folder to:

* Netlify
* Vercel
* GitHub Pages
